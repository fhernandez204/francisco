Spring Boot JPA + H2

Desarrollo de una aplicación que exponga una API RESTful para la creación de usuarios.

REQUISITO:

● Banco de datos en memoria. H2.

● Proceso de build vía Maven.

● Persistencia con JPA. Ejemplo: Hibernate.

● Framework SpringBoot.

● Java 8+
